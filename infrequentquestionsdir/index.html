Q: Why can I view Scratch through the preview box?

A: The preview box is an iframe, which is an element that loads another HTML element inside of a web page. On a PC device, if you wish to avoid this behaviour, hold CTRL (CMD on MacOS) and click to open the URL in a new tab instead of opening it in the preview box.

Q: Why does this URL sign you out?

A: Though this is still technically unanswered (or unconfirmed), the cookie being sent is malformed.
